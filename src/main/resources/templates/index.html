What is the Project for?

This project is intended to demonstrate how a web-service could be created and the software stack it is made up of.

This project is a web service with an API.
The API is very simple and it only has a small data set to keep it understandable.
However it does not persist the data any where and it is envisaged that this service generates its own data.

The service does not draw data from any where it simply generates the data from a random data set.

At the start there will be 50 names in the store and they will be regenerated the next time the service starts.

Full details of the API provided by this application can be seen via an embedded Swagger page.
